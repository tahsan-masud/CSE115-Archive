# Array of Structures (Student Records)

## Description
This program scales up the use of custom data types by managing a collection of student records. It utilizes an array of `Student` structures to store the ID, name, and CGPA of multiple students (up to 100). It takes user input to populate the array and then iterates through it to display the stored data.

## Concepts Covered
* Arrays of structures
* `for` loop iteration
* Global variable declarations
* Memory allocation for strings (`char` arrays)

## How to Run
Compile and run the program:
gcc "structure 100 element array.c" -o student_array
./student_array
