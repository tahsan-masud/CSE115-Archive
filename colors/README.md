# Terminal Colors in C

## Description
A fun exploration of ANSI escape codes in C. This program demonstrates how to manipulate the standard terminal output to display text in various foreground colors, background colors, and text styles (like bold and underline). 

## Concepts Covered
* ANSI Escape Sequences
* Terminal formatting and styling
* Standard I/O operations (`printf`)

## How to Run
Compile the code using GCC and run the executable in a compatible terminal:
gcc colors.c -o colors
./colors
