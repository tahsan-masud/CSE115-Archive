# CSE115: C Programming Fundamentals Archive

A curated collection of my foundational coursework and projects from the CSE115 (Introduction to Programming) course. This repository documents my progression in learning the C programming language and understanding core software development principles.

### Key Concepts Covered Across Projects:
* Standard and File I/O operations
* Dynamic memory allocation (`malloc`, `free`)
* Data structures (`struct` implementation)
* Control flow and sorting algorithms (e.g., Bubble Sort)
* Pointers and array manipulation

### Repository Structure
Each folder contains a specific assignment or project, or just a practice code, along with its own localized `README.md` detailing the problem statement and technical execution.
