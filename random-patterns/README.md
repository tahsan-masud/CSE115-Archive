# Console Pattern Printing

## Description
A classic collection of nested loop exercises designed to build spatial logic. This single file generates multiple complex ASCII shapes in the console, including left-aligned pyramids, hollow diamonds, alphabetic pyramids, and a mathematical generation of Pascal's Triangle.

## Concepts Covered
* Nested `for` loops
* Conditional logic (`if/else`) for hollow shapes
* ASCII character manipulation (`'a' + j - 1`)
* Combinatorics calculation (Pascal's Triangle logic)

## How to Run
Compile and run the program:
gcc "random patterns practice.c" -o patterns
./patterns
