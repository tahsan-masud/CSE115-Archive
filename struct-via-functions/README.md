# Structure Passing by Value

## Description
A foundational exercise in creating custom data types in C. This program defines a `Books` structure to hold information like title, author, subject, and ID. It demonstrates how to populate these structures using user input and pass them to functions by value for printing.

## Concepts Covered
* Structs and `typedef`
* Passing structs to functions (Pass-by-Value)
* Returning structs from functions
* String handling with `scanf(" %[^\n]")`

## How to Run
Compile and run the program:
gcc "passing structure as a function.c" -o struct_pass
./struct_pass
